The "Dictionaries" folder contains the globalization files for VxKex itself.

Items inside KexDir will be installed into KexDir by the installer:

Globalization\
  Dictionaries\
    BDI files for KexMLS to translate user interface strings.
  ICU\
    These files are used by ICU. Not having them will cause errors.

Certificates\
  Contains updated root certificate files for KxSChanl, etc.
